# **👩‍💻👨‍💻 How to Get Started**:

**Clone**

**`$ git clone https://github.com/Chen-Abudi/mini-tic-tac-toe.git`**

**Install**

**`$ cd mini-tic-tac-toe`**

**`$ npm i or npm install`**

**Launch**

**`$ npm run dev`**
